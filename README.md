# Financial Performance Dashboard — Power BI

## Overview
4-page enterprise financial dashboard analyzing $54M revenue 
across 5 departments, 6 product categories and 4 global 
regions covering 2021–2023.

## Pages
1. Executive Summary — KPIs, trends, profit status
2. Revenue Analysis — Waterfall, heatmap, regional breakdown
3. Expense Analysis — Area chart, payment risk, deal size
4. Budget vs Actual — Gauge visual, variance matrix

## Key Features
- Star schema data model with 7 tables
- 3 relationship types: One to Many, One to One, Many to Many
- 14 DAX measures including time intelligence and conditional
- Conditional formatting across all 4 pages
- Dynamic title updating with slicer selection
- Gauge visual for budget tracking

## Tech Stack
Power BI Desktop · DAX · Power Query · Data Modeling
